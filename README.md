# aes256-encrypt
A unix or linux encryption script for using the AES256-CBC cipher using openssl securely and conveniently

CHANGE PERMISSION TO EXECUTABLE
chmod +x encrypt

TO ENCRYPT:
./encrypt -e FILE_NAME

TO DECRYPT:
./encrypt -d FILE_NAME
